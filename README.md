## 🚀 Overview  
This project predicts house prices in Bengaluru using Machine Learning.  
Users can input details like location, area, number of bedrooms, and bathrooms, and receive an estimated property price through a web-based interface.

---

## 🎯 Key Features  
- 🧠 Machine Learning-based price prediction  
- 🌐 Interactive web interface  
- ⚡ Fast API response using Flask  
- 📊 Real-world Bengaluru housing dataset  
- 🔗 End-to-end integration (Frontend + Backend + Model)
